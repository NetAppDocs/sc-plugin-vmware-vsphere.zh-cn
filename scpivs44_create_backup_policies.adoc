= 创建备份策略
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
按照以下说明创建备份策略

. 在SCV插件的左侧“Navigator”(导航器)窗格中，单击“*Policies*”(策略*)，然后选择“*New Policy*”(新建策略*)。
. 在 * 新建备份策略 * 页面上，输入策略配置信息，然后单击 * 添加 * 。
+
如果策略将用于镜像 - 存储关系，则在复制字段中，如果要将备份复制到镜像 - 存储目标，则只能选择 * 备份后更新 SnapVault * 选项。
