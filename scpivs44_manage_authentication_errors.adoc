= 管理身份验证错误
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
如果不使用管理员凭据，则在部署适用于 VMware vSphere 的 SnapCenter 插件后或迁移后，可能会收到身份验证错误。如果遇到身份验证错误，则必须重新启动此服务。

.步骤
. 使用格式 `https://<OVA-IP-address>:8080.` 登录到 SnapCenter VMware 插件管理 GUI
. 重新启动服务。
