=== 其他 ONTAP 信息

* 如果您运行的是 ONTAP 8.2.x ：
+
您必须以 `vsadmin` 的身份登录到 Storage VM 上，以获得适用于 VMware vSphere 的 SnapCenter 插件操作的适当特权。

* 如果您运行的是 ONTAP 8.3 及更高版本：
+
您必须以 `vsadmin` 或具有上表中列出的最低权限的角色登录。
