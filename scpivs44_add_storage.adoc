= 添加存储
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
按照本节中的步骤添加存储。

. 在SCV插件的左侧“Navigator”(导航器)窗格中，单击“*存储系统*”，然后选择“*添加*”选项。
. 在添加存储系统对话框中、输入基本SVM或集群信息、然后选择*添加*。
